# 🌟 PATH ABLE  
### AI Navigation & Safety Assistant for Visually Impaired

---

## 🚀 Project Overview  
**PATH ABLE** is an AI-powered assistive application designed to help visually impaired individuals navigate safely and independently using **real-time guidance, voice interaction, and vibration feedback**.

It acts as a **digital companion**, enabling users to understand their surroundings and take the right actions confidently.

---

## 🎯 Core Idea  
> “It doesn’t just tell what is in front — it tells what to do next.”

Unlike traditional assistive tools, PATH ABLE provides **decision-based navigation**, guiding users with actionable instructions like:  
👉 *“Move slightly right”*  
👉 *“Obstacle ahead, stop”*

---

## ⚙️ Key Features  

### 🧭 Smart Navigation System  
- Detects obstacles in real time  
- Suggests safe directions to move  

### 📳 Haptic Feedback (Vibration Guidance)  
- Uses vibration patterns for direction and alerts  
- Enables navigation even without audio  

### 🎤 Voice Assistant  
- Full hands-free control  
- Audio feedback for instructions and environment  

### 🚨 Smart Emergency System  
- Detects danger situations  
- Automatically sends:
  - 📍 Live location  
  - 📞 Emergency call  
  - 📩 Alert message to family  

### 👁️ Real-Time Object Detection  
- Identifies surroundings like:  
  - People  
  - Walls  
  - Doors  
  - Obstacles
